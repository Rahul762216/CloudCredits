# 📝 Responsive Blog Template using HTML, CSS & JavaScript
A clean and responsive blog website template designed for content creators, writers, and developers. This project allows users to view blog posts in a readable and mobile-friendly layout — perfect for showcasing articles or starting a personal blog.

# 🔐 Features
📰 Blog homepage with multiple posts layout

🧾 Individual blog post pages (optional)

📱 Fully responsive — adapts to mobile, tablet, and desktop

🔍 Search bar or category filter (optional)

🌓 Dark mode toggle (optional)

🖼️ Author name, date, and tags on posts

🎨 Stylish typography and image handling

# 📌 Objective
To create a visually appealing and mobile-friendly blog website template using only HTML, CSS, and JavaScript, suitable for personal blogs, tech articles, or content-based websites.

# 🧰 Technologies Used
HTML5 — Page structure and blog content

CSS3 — Styling, responsive layout, animations

JavaScript (Vanilla) — Interactive elements (e.g., navbar toggle, dark mode, filter)

Optional: You can integrate frameworks like Bootstrap, TailwindCSS, or CMS tools for dynamic content.
